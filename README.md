## Overview:- 

An enterprise-grade system designed to streamline library operations and automate book rental workflows.
Built for scalability, security, and efficiency, offering complete management for books, subscribers,
and rentals with real-time tracking and notifications.


## Key Highlights:-

- Role-based authentication & authorization (Admin, Librarian, Subscriber) using ASP.NET Identity
- CRUD operations for books, authors, subscribers, and rentals
- Automated notifications via Email & WhatsApp using Hangfire
- Background job processing and fine calculation for overdue books
- Integrated Cloudinary for image storage and management
- Advanced search, filtering, and reporting features
- Centralized logging with Serilog and data mapping via AutoMapper


## Tech Stack:- 
ASP.NET Core MVC | EF Core | LINQ | Repository Pattern | Hangfire | Serilog | AutoMapper | Cloudinary | Bootstrap | jQuery | SQL Server
